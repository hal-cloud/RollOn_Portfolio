## ゲーム概要

流れてくるノーツに合わせてボタンを押し、スコアを競うリズムゲームです。  
複数楽曲・難易度選択に対応しており、ハイスコアを記録する機能を備えています。

| サウンド | XAudio2 |
| ライブラリ | DirectXTex, nlohmann/json |

---

## 主な機能・実装

- **ノーツ判定システム** - タイミングに応じた判定（Perfect / Good / Miss など）
- **コンピュートシェーダーによるパーティクル** - GPU側でパーティクルを並列処理
- **オブジェクトプーリング** - ノーツ・エフェクトの動的生成コストを削減
- **コンポーネントシステム** - `GameObject` にコンポーネントをアタッチする設計
- **固定タイムステップのゲームループ** - 120fps 固定で更新処理を安定化
- **楽曲・難易度選択画面** - カルーセルUIで楽曲を選択
- **ハイスコア管理** - プレイ結果を記録・表示
- **フェード・ローディング画面** - シーン遷移演出

## ビルド方法

### 必要環境
- Visual Studio 2022
- Windows SDK

### 手順
1. このリポジトリをクローンする
2. `RollOn.sln` を Visual Studio 2022 で開く
3. ビルド構成を `Release` / `x64` に設定する
4. ビルドを実行する
